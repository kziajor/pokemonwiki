# pokemonwiki

Purpose of this project: Learn Vue + Vuex + Typescript

Based on https://github.com/florinpop17/weekly-projects

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
